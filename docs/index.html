Single Page Web Applications with AngularJS

Johns Hopkins University

Part of a 6-course series, the Ruby on Rails Web Development Specialization

This is the list of implementaions of the course assignments made by Slavoljub Radenkovic.
1. Lunch Checker
This is the assignment of module 1. Simple AngularJS app to get started.
2. Shopping List Check Off
Module 2 assignment, create a "check off" shopping list application.
3. Narrow Down Your Chinese Menu Choice
Module 3 assignment, search the descriptions of menu items, then, given the list of search matches, also give the user the ability to throw the items they for sure don't want off the list, thus narrowing it down to what they do want.
4. Menu App
Module 4 assignment, use the restaurant menu REST API to create a master/detail view pair for catergories and menu items.
5. Newsletter registaration
Module 5 assignment, add a newsletter registaration in an existing app. It's based on the examples/Lecture59
